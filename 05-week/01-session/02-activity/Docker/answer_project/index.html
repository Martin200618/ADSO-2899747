Flight Management System

Geolocation Infrastructure Parameterization Security Human Resources Aircraft Management Flight Operations Passengers & Services Notifications

Presentación del Proyecto

Este sistema gestiona todas las operaciones relacionadas con vuelos: desde la definición de aeropuertos, aviones y tripulación hasta la venta de tiquetes y la notificación de eventos. Incluye módulos para geolocalización, infraestructura, parametrización, seguridad, recursos humanos, gestión de aeronaves, operaciones de vuelo, servicios al pasajero y notificaciones.

participant "Flight Operations\n(Baggage)" as Baggage
participant "Notifications\n(Notification)" as Notification

Passenger -> Person : Provide personal data
Person --> Passenger : Validate identity

Passenger -> Flight : Search available flight
Flight --> Passenger : Return flight details

Passenger -> Ticket : Request booking
Ticket -> Flight : Verify availability
Flight --> Ticket : Availability confirmed
Ticket --> Passenger : Ticket created

Passenger -> Baggage : Register baggage (optional)
Baggage --> Passenger : Baggage confirmed

Ticket -> Notification : Send booking confirmation
Notification --> Passenger : Confirmation message delivered

Tablero de Historias de Usuario (HU)

Geolocation
  • Como Administrador quiero gestionar continentes para administrar ubicaciones globales.
  • Como Administrador quiero asignar estados a países para organizar la información geográfica.
  • Como Administrador quiero agregar ciudades a estados para ubicar los aeropuertos correctamente.
Infrastructure
  • Como Administrador quiero registrar aeropuertos para controlar puntos de origen y destino.
  • Como Administrador quiero definir terminales en cada aeropuerto para organizar embarque y desembarque.
  • Como Administrador quiero definir puertas de embarque para programar los vuelos de manera ordenada.
Parameterization
  • Como Administrador quiero configurar tipos de documentos para capturar correctamente la documentación de las personas.
  • Como Administrador quiero administrar tipos de vuelos para clasificar los vuelos nacionales e internacionales.
  • Como Administrador quiero definir tipos de aeronaves para facilitar la gestión de la flota de aeronaves.
  • Como Administrador quiero definir roles de tripulación para asignar responsabilidades claramente.
  • Como Administrador quiero definir clases de tiquetes para ofrecer diferentes categorías de asientos.
Security
  • Como Administrador quiero administrar roles y permisos de usuarios para controlar el acceso al sistema.
  • Como Administrador quiero registrar usuarios asociados a personas y roles para permitir el inicio de sesión de usuarios.
  • Como Usuario quiero iniciar sesión en el sistema para acceder a funcionalidades según su rol.
Human Resources
  • Como Administrador quiero registrar empleados con sus datos y roles para gestionar tripulación y staff.
  • Como Administrador quiero consultar información de empleados para asignar tripulaciones eficientemente.
Aircraft Management
  • Como Administrador quiero registrar aeronaves con sus tipos y horas de uso para controlar la flota.
  • Como Administrador quiero consultar la disponibilidad de aeronaves para planificar vuelos correctamente.
Flight Operations
  • Como Coordinador quiero programar vuelos con horarios, origen, destino y aeronave para cumplir con la programación de operaciones.
  • Como Coordinador quiero asignar tripulación a un vuelo para garantizar que el vuelo esté atendido.
  • Como Pasajero quiero comprar un ticket para reservar su asiento en un vuelo.
  • Como Pasajero quiero registrar su equipaje para asociar su equipaje con su ticket.
Passengers & Services
  • Como Pasajero quiero registrarse como cliente del sistema para comprar tickets y gestionar sus viajes.
  • Como Pasajero quiero acumular puntos de viajero frecuente para obtener beneficios y recompensas.
Notifications
  • Como Administrador quiero crear plantillas de mensajes para notificar eventos de manera uniforme.
  • Como Sistema quiero enviar notificaciones de confirmación de ticket para mantener informados a los pasajeros.
  • Como Sistema quiero enviar alertas de retraso de vuelo para informar oportunamente a pasajeros y tripulación.
